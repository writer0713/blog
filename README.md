# KJH DEV BLOG

### 사용한 기술 :

- Spring Boot (Java8), JSoup, Thymeleaf, Gradle, Bootstrap, Heroku, Disqus, etc

### 특징 :

- 네이버 블로그 (스마트 에디터)로 글을 쓰면 이 블로그에도 실시간 동기화 된다.
- 원하는 카테고리만 크롤링을 통해 보여준다.
- 구글, 다음 등 여러 검색엔진에 노출 가능하다.
- 광고를 붙이는 등의 커스터마이징이 가능하다.
